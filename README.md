# My Sample Backstage Service

A sample service registered in Backstage to demonstrate the software catalog.

## Features

- REST API for user management
- CI/CD pipeline with GitHub Actions
- TechDocs documentation

## Getting Started

```bash
npm install
npm start
```

## Documentation

View the full documentation in Backstage under the Docs tab.

## API

This service provides the `my-api` REST API. See the API tab in Backstage for the OpenAPI specification.
