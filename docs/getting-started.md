# Getting Started

## Prerequisites

- Node.js 18+
- npm or yarn

## Installation

```bash
npm install my-component
```

## Basic Usage

```javascript
import { MyComponent } from 'my-component';

const component = new MyComponent();
component.init();
```

## Configuration

Create a configuration file `config.json`:

```json
{
  "port": 3000,
  "debug": true
}
```
